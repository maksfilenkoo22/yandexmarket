# yandexmarket
Решение для автоматизированной доставки цифровых товаров на Яндекс Маркет. В данном проекте товарами являются аккаунты для ChatGPT
В проекте вы увидите несколько файлов:
1) worker,py- основной обработчик заказов. Через этот файл товары из вашей созданной БД будут отправляться в доставку на Яндекс Маркет
2) config.py - файл с вашими данными о личном кабинете (id компании, id бизнес, пути к БД и файлу с логами на вашем компьютере)
3) accounts.db - макет БД с аккаунтами (вы можете прямо туда вставить свои товары)
4) worker.log - файл с оповещениями о работе worker.py (количество товаров в общем, сколько товаров получено, сколько товаров отправлено в доставку)
Данный проект работает с личного компьютера, то есть компьютер, где у вас лежит проект, должен быть включен 24\7. Я же автоматизировал работу через сервер.
Вы можете также настроить интеграцию проекта с любым сервером, тогда доставка будет полностью автоматизированной
